# OS_Assignment(SZU)
- 此仓库是本人在操作系统（深圳大学）中的作业，代码仅供参考

--- 
### 包括以下实验
- OS_HW_2： 实验2进程（线程）同步及死锁实验
- OS_HW_3： 实验3内存分配与回收
- OS_HW_Comprehensive_1： 综合实验1-shell
- OS_HW_Comprehensive_2： 综合实验2-ext2

---
### 开发平台及语言
- 语言: C
- 操作系统环境: CentOS7
- IDE: Visual Studio Code
